Show your tweets on your website by "Tweets Slider" plugin. Simple and Light Weighted plugin to publish your tweets on your website.

== Description ==

"tweets-slider" is the very simple and light weighted plugin to show your tweets on your website. "Tweets Slider" has provided you an option to show tweets on your website as slider or in a normal way.

= Features = 
 * An option to define number of tweets
 * An option to define slider speed
 * An option to disable slider
 * Shortcode
 
  Video Tutorial : 
  
  https://youtu.be/k0Iqb7xngvk
 
= Shortcode = 
[tweets_slider]

== Installation ==

Step 1. Upload "tweets-slider" folder to the `/wp-content/plugins/` directory

Step 2. Activate the plugin through the Plugins menu in WordPress

Step 3. Go to Settings/"Tweets Slider" and configure the plugin settings.

== Frequently Asked Questions ==

= Q.1 How enable plugin after install? =

Ans. You can enable plugin from plugin settings page.

= Q.2 How get my access token details? =

Ans. You can access token details from your twitter app account, please browse https://apps.twitter.com/ this url and create your app.

= Q.3 Can display tweets in normal way not as slider ? =

Ans. Yes, You can disable slider option from plugin settings page.
